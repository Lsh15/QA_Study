# TDD
TDD란 Test Driven Development의 약자로 테스트 주도 개발이다.



## 장단점
### 장점
* 튼튼한 객체 지향적인 코드 생산
* 재설계 시간의 단축
* 디버깅 시간의 단축
* 테스트 문서의 대체 가능
* 추가 구현의 용이함

### 단점
* 생산성의 저하
